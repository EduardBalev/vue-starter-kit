# vue-starter-kit
Vue.js + Vue Loader + Webpack + Gulp + Babel

## Introduction
A simple starter for new vue project.

## Quick start
    $ git clone https://github.com/zhuowenli/vue-starter-kit.git
    $ cd vue-starter-kit
    $ npm i
    $ npm start
